Poker Dice Stratagem

Poker Dice Stratagem is a fast 4‑player dice poker variant: everyone rolls simultaneously in leather cups, keeps 3 dice, donates 2 to a communal pool, the pool is rerolled through a wooden dice tower, then everyone drafts back two dice in a snake pick. The winner scores by rarity.

Components

ItemQuantity
Players4 (locked)
Classic d6 dice5 per player (20 total)
Leather dice cups1 per player (4 total)
Wooden dice tower1 (communal reroll)
Wooden hexes (score tokens)400 in a velvet bag
Start Token1 (any marker)

Objective

Be the first player to reach 100 points. Points are tracked using wooden hexes drawn from the velvet bag. If you stop due to time, the player with the highest score wins.

Setup

  1. Put all 400 wooden hexes into the velvet bag and place it in the center.
  2. Choose a starting player once (any method you like).
  3. Give that player the Start Token.

Round flow

Round mantra: Roll → Keep 3 → Offer 2 → Tower reroll → Snake draft 2 → Showdown → Score

1) Cup Roll (simultaneous)

All players roll 5 dice at the same time using their cups.

2) Keep 3 / Donate 2 (simultaneous)

Each player chooses exactly 3 dice to keep, and places the remaining exactly 2 dice into the center pool.

3) Tower Reroll (communal)

Roll the 8 donated dice (2×4) through the dice tower. Place the results in the center as the Draft Pool.

4) Snake Draft (each player drafts 2 dice)

Starting with the player holding the Start Token, draft dice from the Draft Pool in this exact order: 1 → 2 → 3 → 4 → 4 → 3 → 2 → 1. After drafting, each player has 5 dice again.

Hands and scoring (unified ladder)

The highest hand wins the round and scores the listed points. Only the round winner scores. Scatter means all five values are different and it is not a straight.

Tie-breakers

If players have the same hand type, break ties quickly using the rules below.

Hand typeTie-break rule
Five / Four / Three of a kind Higher matching set wins (e.g., four 6s beats four 5s). If still tied, compare remaining dice high→low.
Full House Higher trip wins; if tied, higher pair wins.
Two Pair Higher top pair, then higher second pair, then kicker.
Straight 23456 beats 12345.
Scatter / One Pair Compare dice sorted high→low.

Exact ties: shared payout

If players are still tied exactly after tie-breakers (same dice values), they share the payout. Let P be the points for the tied hand and T the number of tied players. Each tied player scores ceil(P / T) and draws that many wooden hexes from the velvet bag.

Example: Three-way exact tie on Scatter (5 points) → 5/3 = 1.67 → each tied player takes 2 hexes.

End of round

  1. The winner (or tied winners) draws hexes from the velvet bag and adds them to their score pile.
  2. Pass the Start Token clockwise.
  3. Begin the next round.

Game end

  • The first player to reach 100+ points wins immediately, or
  • If time is called, highest score wins.
