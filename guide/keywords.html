Keyword Icons

Icons are loaded from assets/icons/. Any text inside braces like {U} or {Vigilance} is rendered as an icon if a matching PNG exists.

Filename Mapping

  • {U} → assets/icons/U.png
  • {W} → assets/icons/W.png
  • {UB} → assets/icons/UB.png
  • {T} → assets/icons/T.png
  • {Tn} → assets/icons/Tn.png
  • {Vigilance} → assets/icons/Vigilance.png

Tips

  • Filenames are case-sensitive on GitHub Pages. Tn.png is not the same as tn.png.
  • Prefer simple tokens (no spaces) inside braces. Example: {FirstStrike} instead of {First strike}.
  • If you see missing icons, confirm the exact token text in the CSV matches the PNG filename.

If you want, this page can auto-generate a keyword list from your CSV later — but first, let the UI stop trying to time-travel.
